# Script-Speedtest.cn-AdBlock-BeautifyLayout
油猴脚本-Speedtest.cn页面净化/屏蔽广告/美化--支持移动端页面
# -功能介绍
### 去广告
去除页面顶部推广
去除页面两侧弹窗
去除页面底部推广
### 优化布局
去除需要安装APP的节点
修复去除后界面布局
